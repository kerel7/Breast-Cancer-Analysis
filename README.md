# Breast Cancer Diagnostic Data Analysis

## Objective
To explore and model the relationship between tumor morphometric features and breast cancer diagnosis.

## Dataset
Breast Cancer Wisconsin (Diagnostic) dataset from UCI Machine Learning Repository.

## Methods
Exploratory data analysis, non-parametric statistical testing, and baseline logistic regression.

## Key Results
- Significant differences in radius_mean between diagnostic groups
- Logistic regression achieved ROC-AUC ≈ 0.99

## Reproducibility
Run the notebook in `01_data_loading_and_cleaning.ipynb`.
